Classpublic class LegendSettings
InheritanceLegendSettings flash.events.EventDispatcher

Legend settings. Will be applied to all legends.

If a property has no default value and it is not defined, default styles and properties of StockLegend will be used.

You can set legend-specific properties and styles directly on StockLegend only if a property doesn't have default value. Note, property set here is of a higher priority.
