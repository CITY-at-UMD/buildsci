Packagecom.ammap.mapClasses
Classpublic class MapMovie
InheritanceMapMovie MapObject flash.events.EventDispatcher

MapMovie is a class where properties of movies are defined. MapMovie can be:
external swf or image file;
class;
component;
predefined movie (circle, rectangle or bubble).

MapMovie can also display labels.

In case some property is not defined in this class, MapMovie will use MoviesSettings class property with the same name.

widthAndHeightUnits:String  [read-write]

You can set miles or kilometers as a unit of width/height of a movie. This works only with predefined movies (type is set). rightLongitude and bottomLatitude should be set with real values in order this to work properly. fixedSize will be set to false if miles or kilometers is set here.

The default value is pixels.
