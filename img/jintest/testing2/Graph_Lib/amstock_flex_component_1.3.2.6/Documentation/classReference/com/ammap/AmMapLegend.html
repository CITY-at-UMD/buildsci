Classpublic class AmMapLegend
InheritanceAmMapLegend mx.containers.Tile

AmMapLegend is a class which displays legend on the map. It's a simple extension of a Tile component and it does not have any direct interaction with the map, as there can be too many different interaction scenarios. So in case you want the legend to do some actions, you should add event listeners and create your own custom methods.

You can add this legend to any container. If you want to add it on top of a map, add both map and the legend to the Canvas and specify legend x and y position. Or, you can place a legend on top of a map as any other component - by adding MapMovie to the DataSet and setting component="yourLegendId".
