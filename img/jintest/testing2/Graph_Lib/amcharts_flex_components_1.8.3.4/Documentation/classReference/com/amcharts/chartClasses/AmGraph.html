Packagecom.amcharts.chartClasses
Classpublic class AmGraph
InheritanceAmGraph mx.core.UIComponent

Graph class displays all types of graphs - line, column, step line, smoothed line, ohlc and candlestick. XY and Radar charts can only display "line" type graphs.

View the examples.
