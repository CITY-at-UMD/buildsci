Classpublic class CategoryAxesSettings
InheritanceCategoryAxesSettings flash.events.EventDispatcher

Category axis settings. Will be applied to all category axes.

If a property has no default value and it is not defined, default styles and properties of CategoryAxis will be used. Check AmChartsBundle documentation to see these defaults.

You can set axis-specific properties and styles directly on CategoryAxis only if a property doesn't have default value. Note, properties set here have higher priority.

dateFormats:Array  [read-write]

Array of date format objects. Date format object must have "period" and "format" items. Available periods are: fff - millisecond, ss - second, mm - minute, hh - hour, DD - date, MM - month, YYYY - year. format can contain any string supported by mx.formatters.DateFormatter class. For example: {period:"DD", format:"MMM DD"}

Maximum series shown at a time. In case there are more data points in the selection than maxSeries, the chart will group data to longer periods, for example - you have 250 days in the selection, and maxSeries is 150 - the chart will group data to weeks.
