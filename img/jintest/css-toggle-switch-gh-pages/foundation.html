CSS Toggle Switches

Accessible, screen-reader friendly, flexible, CSS-only toggle switches with full keyboard access and mobile support, using standard form controls.

These examples use Foundation, you can also use the toggles alone, or with Bootstrap.

Download

The toggle switches are very flexible, so you can use a number of classes provided by Foundation, along with it's grid, to make them look and act exactly as you need them.
