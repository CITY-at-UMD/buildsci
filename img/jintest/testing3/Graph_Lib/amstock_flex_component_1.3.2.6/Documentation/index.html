Installing components

To use SWC files in your Flex projects, you should add them to the project's library path.

  1. With a project selected in the Flex Navigator view, select Project > Properties > Flex Build Path.
  2. Click on the Library Path tab.
  3. Click on the Add Swc button.
  4. Browse to and select the amStockFB3.swc file if you work with Flex Builder 3. Select amStockFB4.swc if you work with Flash Builder 4 (aka Gumbo).
  5. Click OK
  6. The component is now added. You should see amCharts folder in the components window (only available when you are in design mode).
Visit Flex documentation for more options on installing components.

1.0.5.0

This is important update!

Some changes which might affect previous behavior were made in this version, read the following carefully.

AmLegend Class no longer extends Canvas, it extends UIComponent instead. This means that AmLegend doesn't support styles/properties/methods specific to Canvas anymore. It still has styles backgroundColor, backgroundAlpha, borderColor and borderAlpha. This change might cause error on your application if you used Canvas specific styles or properties. If you need them anyway, you can always put our legend into Canvas container.

Bug fixes:

sometimes zoomed event was fired multiple times.

when data sets were compared, and size of a window changed, the chart changed base date.

when recalculateToPercents was set to "always", the legend still displayed absolute values
