Classpublic class ValueAxesSettings
InheritanceValueAxesSettings flash.events.EventDispatcher

Value axis settings. Will be applied to all value axes.

If a property has no default value and it is not defined, default styles and properties of ValueAxis will be used. Check AmChartsBundle documentation to see these defaults.

You can set axis-specific properties and styles directly on ValueAxis only if a property doesn't have default value. Note, property set here is of a higher priority.
