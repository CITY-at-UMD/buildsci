openPeriodValue:String  [read-write]

openPeriodValue is used in a quite rare scenario only: in case you have openField set for line/step/column graph types and when grouping data different period value than set in periodValue property needs to be used (min for example), then you have to setit here. Possible values are: Open, Low, High, Close, Average and Sum

The default value is undefined.
