Packagecom.ammap.mapClasses
Classpublic class MapObject
InheritanceMapObject flash.events.EventDispatcher
SubclassesDataSet, MapArea, MapLine, MapMovie

MapObject is a base class of MapMovie, MapLine, MapArea and DataSet. Note, MapObject is not a DisplayObject of the object. To access the DisplayObject, use MapArea.displayObject property.

urlproperty 
url:Object  [read-write]

url of an object. url of an object can be a real URL (string) - the application will go to this url when you click on the object. Note, this might work only if you upload the application to web server (for security restrictions).

It can also be id of other MapObject - MapArea, MapMovie, MapLine or DataSet. Then user clicks on this object the application would zoom-in to this objects' zoom position (if it is set) and then act as if the object specified in the url was clicked.

This can be used for selecting the object you want to be selected right after the map is initialized, as DataSet extends MapObject class, you can specify url for your DataSets.
