Classpublic class AmBalloon
InheritanceAmBalloon mx.core.UIComponent

AmBalloon is the class which generates balloons (datatips). Balloon follows the mouse when you roll-over the pie slice, chart indicator of Serial charts displays value balloons and category balloon. Chart shows/hides and sets position for every balloon automatically, so all you need to do is to change balloon appearance using styles.

View the examples.
