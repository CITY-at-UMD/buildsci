Packagecom.ammap
Classpublic class ObjectList
InheritanceObjectList mx.containers.Canvas

Object list displays the list of MapObjects available in selected DataSet. It can be displayed as a Tree, ComboBox or List.

DataSet class has showAreasInList, showLinesInList and showMoviesInList properties which specify if areas, lines and movies should be included in the ObjectList. MapObject has showInList property which allows to exclude from the list any area, line or movie.
