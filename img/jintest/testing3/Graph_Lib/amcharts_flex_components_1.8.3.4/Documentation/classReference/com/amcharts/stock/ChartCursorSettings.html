Classpublic class ChartCursorSettings
InheritanceChartCursorSettings flash.events.EventDispatcher

Chart cursor settings. Will be applied to cursors of all charts.

If a property has no default value and it is not defined, default styles and properties of ChartCursor will be used. Check AmChartsBundle documentation to see these defaults.

You can set chart-specific properties and styles directly on ChartCursor classes only if a property doesn't have default value here. Note, properties set have higher priority.

categoryBalloonDateFormatsproperty 
categoryBalloonDateFormats:Array  [read-write]

Array of date format objects. Date format object must have "period" and "format" items. Available periods are: fff - millisecond, ss - second, mm - minute, hh - hour, DD - date, WW - week, MM - month, YYYY - year. format can contain any string supported by mx.formatters.DateFormatter class. For example: {period:"DD", format:"MMM DD"}

The default value is [{period:"YYYY", format:"YYYY"}, {period:"MM", format:"MMM DD, YYYY"}, {period:"WW", format:"MMM DD, YYYY"}, {period:"DD", format:"MMM DD, YYYY"}, {period:"hh", format:"JJ:NN"},{period:"mm", format:"JJ:NN"}, {period:"ss", format:"JJ:NN:SS"}, {period:"fff", format:"JJ:NN:SS"}].

cursorPositionproperty 
cursorPosition:String  [read-write]

Position of a cursor. Specifies where the cursor line should be placed - on the beginning of the period (day, hour, etc) or in the middle (only when parseDates property of categoryAxis is set to true). If you want the cursor to follow mouse instead glueing to the nearest data point, set "mouse" here. Possible values are: start, middle, mouse.
